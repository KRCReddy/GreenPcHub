# LaptopHub - Clean version

This is a clean, deploy-ready scaffold (Vite + React frontend, Express + TypeScript backend, SQLite).
- Build: `npm ci && npm run build`
- Start: `npm start`
- Admin: set env `VITE_ADMIN_PASSWORD` to a strong password.

Deploy to Render as a single Web Service (root), Build Command: `npm ci && npm run build`, Start Command: `npm start`.
